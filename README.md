# VIVO — Biblia para Jóvenes v1.0

Aplicación PWA católica para jóvenes de 16-30 años con IA pastoral responsable.

## 🚀 Deploy en Vercel

1. Conectar repositorio GitHub
2. Agregar variable: `ANTHROPIC_API_KEY`
3. Deploy automático

## ✨ Características

- ☀️ Evangelio del día (Vatican News)
- 🔥 Temas para jóvenes (9 módulos)
- 📖 Estudios bíblicos (8 fundamentales)
- 🙏 Chat con Padre Tomás (IA pastoral)
- 🎨 Wallpapers de Salmos

## 🛠 Stack

- React 18 + Vite 5
- Anthropic Claude API
- PWA completa
- Biblia CEE 2011

## 📱 PWA

La app es instalable como aplicación nativa en móvil.
